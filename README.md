# ETL / DWH QA Assessment Repository

This repository is structured for **real-time ETL/DWH QA interview and project demonstrations** in a professional format commonly used in TCS / Infosys / Accenture style assessments.

## Repository Structure
- `data/` → source Excel and raw input files
- `docs/` → requirement analysis, test strategy, RTM
- `sql/` → validation SQL queries
- `test-cases/` → scenario → test case → expected result
- `defects/` → sample defect logs
- `.github/workflows/` → optional CI checks for SQL lint / file validation
